# Sleep-Alters-Neurovascular-and-Hydrodynamic-Coupling-in-the-Human-Brain

This repository contains script used in research article "Sleep Alters Neurovascular and Hydrodynamic Coupling in the Human Brain"
Currently available at Biorxiv: doi: https://doi.org/10.1101/2024.10.16.618430 

The analysis requires following external packages:
PhaseTE calculation; Nitin Williams, eeg_meg_analysis, GitHub
https://github.com/nitinwilliams/eeg_meg_analysis/tree/master/phaseTE

Morlet wavelet convolution adapted from;
Mike X Cohen, AnalyzingNeuralTimeSeries, GitHub
https://github.com/mikexcohen/AnalyzingNeuralTimeSeries

Color packages:
SlanCM: https://github.com/slandarer/slanColor/tree/main
ColorBrewer: https://se.mathworks.com/matlabcentral/fileexchange/45208-colorbrewer-attractive-and-distinctive-colormaps
